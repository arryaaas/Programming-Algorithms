# Hello World

Program sederhana untuk menampilkan teks *"Hello World!"* ke layar menggunakan bahasa C++.

```cpp
#include <iostream>

using namespace std;

int main() {
    cout << "Hello World!";
    return 0;
}
```

Keterangan :
1. `cout <<` digunakan untuk menampilkan output ke layar.
2. `return 0` mengembalikan nilai kepada fungsi main.
